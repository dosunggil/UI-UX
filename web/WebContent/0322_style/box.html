박스 모델

박스모델이란 웹 문서내용을 박스 형태로 표현하는 것을 말한다

나열형태에 따라 블록과 인라인이 있다

블록 : 태그 하나가 너비 100%를 차지

인라인 : 너비를 태그의 컨텐츠만큼 차지


블록(block) div, p, form, hr, table, ul, ol, h1~h6
인라인(inline) span, strong, label, a : inline
input, textarea, button, img, : inline-block

박스모델의 속성

  • content : 텍스트나 이미지
  • border : 테두리
  • padding : 안쪽 여백으로 border와 content 사이의 영역
  • margin : 바깥쪽 여백으로 태그와 태그 사이의 영역

inline 과 inline-block 비교하기

* inline 은 width, height가 적용되지 않고 컨텐츠 만큼 크기를 차지한다.
* inline-block 은 width, height를 적용하면 그만큼 크기가 변화한다.

box-sizing 관련 : content-box와 border-box 비교하기

content-box : 컨텐츠 영역만 width, height 에 해당하고, padding, border 는 포함되지 않는다.

border-box : 컨텐츠 영역 + padding + border 까지가 width, height에 해당
