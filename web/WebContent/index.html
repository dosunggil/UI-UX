1.블록 레벨 텍스트 태그

:혼자서 한 줄을 차지하여 너비가 100% 꽉 차게 표시되는 태그

1)h태그: heading태그 - 제목을 표시하는 태그로 h1~h6

h1 : 가장큼

h6 : 가장작음

2)p태그: paragraph - 단락을 표시하는 태그로 줄바꿈 없이 텍스트가 한 줄로 표시됨
텍스트의 길이가 브라우저 창의 너비보다 길면 자동으로 줄이 바뀐다.

3) br태그: 줄바꿈 태그로 끝태그가 없다

4) hr태그(horizontal): 주제가 바뀌거나 할 떄 가로선 표시
끝태그가 없다


5) div태그(division) : 영역을 나누는 태그


2. 인라인 레벨 텍스트 태그

한 줄을 차지하지 않는 태그로 화면에 표시되는 컨텐츠 만큼의 너비를 차지하는 태그

1) strong태그 : 텍스트를 굵게 표시하는 태그

굵은 텍스트

자동완성 : ctrl+space
ctrl+alt+ 방향키 위아래 : 지정된 문단을 위 또는 아래로 복사 붙여넣기 한다
ctrl+d : 지정된 문단 삭제
alt+방향키 위아래 : 지정된 문단 이동

2) span태그 : 특별한 뜻은 없지만 텍스트의 일부만 스타일 적용

텍스트 일부만 스타일 적용하기

4. 표를 만드는 태그

table태그, tr태그(table row), td태그(table data)
행/열로 이루어진 표의 형태를 만들때 사용하는 태그
tr: 행에 해당하는 태그
td: 행 내부의 열에 해당하는 셀 태그
th:(table head): 행 내부의 제목에 해당하는 셀 태그
스타일은 table 옆에 쓴다 / border
