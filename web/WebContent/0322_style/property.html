속성 선택자

선택자[속성] : 해당 속성을 갖는 모든 요소를 선택

선택자[속성=값] : 해당 속성의 해당 값을 갖는 모든 요소를 선택

선택자[속성!=값] : 해당 속성의 해당 값을 갖지 않는 모든 요소를 선택

선택자[속성^=값] : 해당 속성이 해당 값으로 시작하는 모든 요소를 선택

선택자[속성$=값] : 해당 속성이 해당 값으로 끝나는 모든 요소를 선택

선택자[속성*=값] : 해당 속성 중 해당 값이 포함된 모든 요소를 선택

선택자[속성~=값] : 해당 속성 중 해당 값이 단어로 포함된 모든 요소를 선택
